------------------------------------------------------------------------------
LINUX
------------------------------------------------------------------------------
System requirements:
* SDL library (version 1.2 or later)
* OSS library (or ALSA with OSS emulation)
Unpack archive with SunVox. 
Run sunvox.



------------------------------------------------------------------------------
WINDOWS
------------------------------------------------------------------------------
Unpack archive with SunVox. 
Run sunvox.exe



------------------------------------------------------------------------------
WINDOWSCE (WINDOWS MOBILE)
------------------------------------------------------------------------------
Copy sunvox.cab to your mobile device. 
Install it.
Now SunVox is in the /Program Files/SunVox folder of your device.
Open Windows start menu. Click on SunVox.



------------------------------------------------------------------------------
PALMOS
------------------------------------------------------------------------------
Unpack archive with SunVox. 
Install sunvox.prc and font.pdb files to Palm.
WARNING. SunVox may crash some new devices like Treo 700p. In this case you
         must install sunvox_for_new_devices.prc instead of sunvox.prc.
Copy other files (examples, samples) to the Flash card.
Run sunvox.
