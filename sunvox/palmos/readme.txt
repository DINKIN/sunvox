sunvox.prc - normal version for most PalmOS 5.x devices.
sunvox_framebuffer.prc - use this version in case of problems with the screen.
sunvox_config.ini - if you want to customize some SunVox options, edit this file and put it to the root of the SD card of your PalmOS device.

Note: This application is not compatible with PiLoc on some devices. Disable PiLoc in case of crash.
Note2: If your 5-way navigation pad is not working, add scan_hard_buttons option to the sunvox_config.ini, and then put this file to the root of the SD card.
